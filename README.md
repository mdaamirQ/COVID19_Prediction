# SEIRD  Model:
* **S**usceptible (can be infected, 'Healthy')
* **E**xposed (infectious but can't spread disease, 'Healthy'
* **I**nfected (infectious and can spread disease)
* **R**ecovered (were infected, immnune to the disease)
* **D**eath (deceased)
